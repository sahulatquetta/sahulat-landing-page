**سہولت (Sahulat) - ہوم سروسز لینڈنگ پیج**

یہ پروجیکٹ 'سہولت' کے لیے ایک جدید اور ریسپانسیو لینڈنگ پیج ہے، جسے کوئٹہ اور پاکستان بھر میں ہوم سروسز کی فراہمی کو آسان بنانے کے لیے ڈیزائن کیا گیا ہے۔

**ہماری خدمات:**

ہم درج ذیل ہوم سروسز فراہم کرتے ہیں:

* الیکٹریشن (Electrician)
* پلمبر (Plumber)
* واشنگ مشین ریپئر (Washing Machine Repair)
* کارپینٹر (Carpenter)
* بائیک مکینک (Bike Mechanic)
* موٹر مکینک (Motor Mechanic)
* اے سی ریپئر (AC Repair)
